# 超级抢购助手

## 功能说明

- 自动帮找隐藏优惠券，有优惠券跳到优惠券界面，无优惠券跳到购买页面。
- 实现多平台电商链接口令一件跳转，复制电商链接口令，点击抢购捷径自动跳转到相应的购买页面，提高你的抢购效率。
- 一般抢购步骤：复制商品链接 → 打开「京东」电商 app→ 打开客服对话框 → 粘贴商品链接 → 发送商品链接 → 对话框点击商品链接跳转抢购页面
- 超级抢购助手步骤：复制商品链接 → 打开超级抢购助手 → 自动跳转抢购页面。
- 目前支持淘宝、京东、苏宁、拼多多等平台。
