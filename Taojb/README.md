# 淘金币自动任务

## 功能说明

- 实现简单的淘金币自动任务
- 兼容 ios13-15 多终端 多账号 自动化
- ⚠️ 注意:因各种权限原因，本快捷指令只能完成部分任务，剩余任务与自己看情况手动，作者会尽力往全自动更新，请关注新版本。
